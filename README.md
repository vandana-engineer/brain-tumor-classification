
# Brain Tumor Classification

## Overview
A Deep Learning project that classifies MRI brain scan images and detects brain tumors using TensorFlow and Computer Vision techniques.

## Features
- MRI image preprocessing
- Brain tumor classification
- TensorFlow CNN model
- Flask web application
- Prediction pipeline
- Modular project structure

## Tech Stack
- Python
- TensorFlow
- OpenCV
- NumPy
- Flask
- Scikit-Learn
